# Before & After: Tutorial Button Naming

## 🔄 Visual Comparison

### **BEFORE (Confusing)**

```
┌──────────────────────────────────────────────────────────┐
│  Entity & Filings  [📹 Watch Tutorial] [❓ Show Tutorial] │
└──────────────────────────────────────────────────────────┘
                            ↑                    ↑
                      What does this      What does this
                      show exactly?       show exactly?
                      
❌ Problem: Both buttons suggest "showing" content
❌ Problem: Users can't distinguish between the two
❌ Problem: "Watch" and "Show" feel redundant
```

### **AFTER (Clear)**

```
┌──────────────────────────────────────────────────────────┐
│  Entity & Filings  [📹 Video Guide] [📖 Quick Start]      │
└──────────────────────────────────────────────────────────┘
                            ↑                  ↑
                    Watch a video      Get interactive
                    explanation        step-by-step help
                    
✅ Solution: Clear distinction in purpose
✅ Solution: Users know exactly what to expect
✅ Solution: Professional SaaS naming convention
```

## 💡 Semantic Analysis

### **Word Association Test:**

**"Show Tutorial"**
- Show = display, reveal, demonstrate
- Tutorial = learning, instruction
- Combined meaning: "Display some tutorial" (vague)
- User expectation: ??? (unclear)

**"Quick Start"**
- Quick = fast, immediate, efficient
- Start = begin, initiate, launch
- Combined meaning: "Get started quickly"
- User expectation: Immediate actionable help ✅

**"Watch Tutorial"**
- Watch = view, observe, see
- Tutorial = learning, instruction
- Combined meaning: "View a tutorial" (could be anything)
- User expectation: Probably a video (not guaranteed)

**"Video Guide"**
- Video = moving images, visual content
- Guide = instruction, walkthrough, help
- Combined meaning: "Instructional video"
- User expectation: Definitely a video tutorial ✅
