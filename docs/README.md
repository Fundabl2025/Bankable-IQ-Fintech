# BANKABLE IQ — Strategic Documentation

This `docs/` folder is the canonical source of truth for the BANKABLE IQ platform's strategy, engineering blueprint, design, and build playbooks.

## Folder structure

| Folder | Contents |
|--------|----------|
| `blueprint/` | The strategic constitution. BANKABLE Corporate Blueprint v1.8. |
| `strategy/` | The reconciliation analysis and spec patch v1.1 that translate the blueprint into engineering direction. |
| `spec/` | The Master Engineering Build Specification v1.0. |
| `migration/` | Migration reports analyzing this existing codebase against the new spec. |
| `playbooks/` | Step-by-step build playbooks (v1, v2.1, v2.2, v2.3). |
| `design/` | Figma file references and design system documentation. |

## Read order for new team members

1. `blueprint/BANKABLE-Blueprint-v1.8.md` — what we are building strategically
2. `strategy/BANKABLE-IQ_RECONCILIATION-CORRECTED_v1.8-aware.md` — how the strategic, engineering, and code layers align
3. `spec/engineering-spec-v1.0.html` — the engineering implementation guide
4. `strategy/BANKABLE-IQ_SPEC-PATCH-v1.1_AND_BOOTSTRAP-RIDER.md` — the v1.8-aligned engineering deltas + fleet bootstrap instructions
5. `playbooks/BANKABLE-IQ_AUTONOMOUS-BUILD-PLAYBOOK_v2.1.md` — the 27-agent autonomous fleet model
6. `playbooks/BANKABLE-IQ_OPERATOR-COMPANION_v2.2.md` — Console, demos, decisions, archeologist, backup
7. `playbooks/BANKABLE-IQ_VISUAL-FOUNDATION-SPRINT_v2.3.md` — the visual constitution Block 1 + Block 2
8. `design/FIGMA.md` — the Figma brand foundation file
